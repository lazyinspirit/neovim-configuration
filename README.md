Lightweight plug n play neovim configuration for editing python/html/javascript/c/c++ scripts
Step 1: Download it in the ~/.config directory via 'git clone https://github.com/lazyinspirit/neovim-configuration/edit/main/README.md' 
Step 2: Rename to nvim
Step 3: run 'nvim' in the terminal, it will auto install the lazy.nvim package manager and set everything up
